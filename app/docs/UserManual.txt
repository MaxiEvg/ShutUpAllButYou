
SUABY — РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ

Назначение:
Утилита позволяет по нажатию горячей клавиши заглушать звук всех приложений, кроме активного окна.

Использование:
1. Убедитесь, что рядом находятся svcl.exe и nircmd.exe
2. Запустите SUABY.jar
3. При первом запуске установите сочетание клавиш
4. При необходимости нажмите «Переместить в трей»
5. Клик средней кнопкой мыши по иконке в трее — активация функции
6. Просмотр логов — в файле suaby.log
7. Изменение настроек — через GUI

Требования:
- Windows 10/11
- Java 17+
- Разрешение на выполнение PowerShell-скриптов
